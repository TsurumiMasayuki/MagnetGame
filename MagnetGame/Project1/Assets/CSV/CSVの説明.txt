0 = 何もない
1 = 通常ブロック
o = 物体N

物体N:
o|L
2個目 = 右R, 左L, B:ブーツ

大きいブロック:
g~でグループ指定をする。
同じグループのブロックと結合する。

通常ブロックの場合
g~|1
1個目 = グループ指定
2個目 = ブロックの指定

磁石の場合
g~|N|0|0
1個目 = グループ指定
2個目 = 極の指定
3個目 = 動く磁石かどうか 0 = true 1 = false
4個目 = 極を変えられるかどうか 0 = true 1 = false


マップサイズ参考
セルサイズが32x32の場合、40x23のマップを作ればだいたい画面一個分のマップができます。
